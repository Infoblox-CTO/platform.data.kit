# Demo: Dev Lifecycle
# Shows: dp dev up → dp dev status → dp dev down
# Prerequisites: dp binary (make build), k3d cluster, DP_E2E_DEV=1

REQUIRE: k3d
REQUIRE: docker
REQUIRE: DP_E2E_DEV

SAY: Welcome to the Data Platform dev lifecycle demo

SAY: This demo shows the local development stack lifecycle
SAY: We'll deploy Helm charts to k3d, check status, and tear down

SAY: Starting the local development stack
CMD: dp dev up

WAIT: 2

SAY: Let's check the status of all services
CMD: dp dev status

WAIT: 1

SAY: All services are running. Now let's tear everything down
CMD: dp dev down

SAY: The local development stack has been cleaned up.
SAY: That's the complete dev lifecycle: up → status → down
