Welcome to the Data Platform! This section will help you get up and running with the `dp` CLI in under 30 minutes.

1. **Set up your environment** with the required prerequisites
2. **Install the DP CLI** on your machine
3. **Create and run your first pipeline** using the quickstart tutorial

## What You'll Learn

By the end of this guide, you'll be able to:

- ✅ Install and configure the `dp` CLI
- ✅ Create a new data package with `dp init`
- ✅ Start the local development stack with `dp dev up`
- ✅ Validate your package with `dp lint`
- ✅ Run your pipeline locally with `dp run`
- ✅ Build and publish your package with `dp build` and `dp publish`
- ✅ Promote your package to an environment with `dp promote`
